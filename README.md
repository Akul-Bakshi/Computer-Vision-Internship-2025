# Computer-Vision-Internship-2025

A collection of projects completed during an AI and Computer Vision internship covering image processing, deep learning, object detection, object tracking, automatic number plate recognition (ANPR), model optimization, and a full-stack facial recognition attendance system.

The repository demonstrates the progression from fundamental OpenCV operations to advanced real-time computer vision applications using YOLO, CNNs, OCR, object tracking, deployment optimization, and facial recognition technologies.

- MongoDB Integration

- Real-Time Web Dashboard

- RESTful APIs
- Attendance Analytics

### Technologies

- MongoDB
- OpenCV
- YOLOv8n-Face
